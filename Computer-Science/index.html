Original Photo by Austin Chu. Right: original image. Left: image after DoG filter.

Image stylization using the Difference of Gaussians (DoG) algorithm. DoG can be used for a variety of image stylization techniques, and here is it set up for edge detection. I implemented the algorithm using Java.

Two Not Touch puzzle generator

Official 10x10 Two Not Touch puzzle

Randomly generated 10x10 puzzle

My favorite puzzle game is Two Not Touch by krazydad, also called Star Battle Go. I downloaded an app with puzzles on my phone, but eventually ran out. Instead of buying more puzzles, I decided I would try to build a program to generate them automatically. It was a lot of work, but I eventually succeeded in making an algorithm that could generate any square two not touch board. You can see the command line output displaying a generated 10 by 10 board on the right. I hope to someday expand this into an app with a real GUI, and maybe even build an automatic solver.
